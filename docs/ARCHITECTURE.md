# HWP Platform

## Versão

2.1 - Foundation

---

# Objetivo

O HWP Platform é um aplicativo PWA para acompanhamento de emagrecimento com preservação e ganho de massa muscular.

Seu foco é oferecer uma experiência simples, rápida e inteligente, utilizando o ChatGPT como principal motor de análise nutricional através do padrão HWP_FOOD.

---

# Princípios

- Poucos toques
- Interface limpa
- Offline First
- Sem APIs pagas
- Dados consistentes
- Código modular
- Arquitetura escalável

---

# Módulos

- Dashboard
- Diário
- Alimentação
- Biblioteca
- Treinos
- Tirzepatida

# Core

O Core é composto por:
